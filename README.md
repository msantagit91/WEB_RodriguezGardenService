# 🌿 Rodriguez Garden Service

Aplicación web desarrollada con **Next.js**, **React**, **TypeScript**, **Tailwind CSS** y **Supabase**, diseñada para digitalizar la gestión operativa de una empresa de jardinería profesional.

El proyecto combina una **landing page moderna**, un **panel administrativo privado**, gestión de clientes y servicios, un **calendario inteligente**, integración con base de datos en tiempo real y despliegue en **Vercel**, aplicando buenas prácticas de desarrollo Full Stack y una experiencia de usuario enfocada en dispositivos móviles.

---

# 🏷️ Tecnologías y Herramientas

* Next.js
* React
* TypeScript
* Tailwind CSS
* Supabase
* Resend
* Vercel
* Git
* GitHub
